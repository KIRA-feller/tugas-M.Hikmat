Hikmat a.k.a Kira Yoshikage


Nama saya Yoshikage Kira. Saya berusia 33 tahun. Rumah saya di bagian timur laut Morioh, tempat semua vila berada, dan saya belum menikah. Saya bekerja sebagai karyawan di department store Kame Yu, dan saya pulang setiap hari paling lambat pukul 20.00. Saya tidak merokok, tetapi terkadang minum alkohol. Saya sudah tidur pukul 23.00, dan memastikan saya tidur selama delapan jam, apa pun yang terjadi. Setelah minum segelas susu hangat dan melakukan peregangan sekitar dua puluh menit sebelum tidur, saya biasanya tidak mengalami masalah tidur hingga pagi. Seperti bayi, saya bangun tanpa rasa lelah atau stres di pagi hari. Saya diberitahu tidak ada masalah pada pemeriksaan terakhir saya. Saya mencoba menjelaskan bahwa saya adalah orang yang ingin menjalani kehidupan yang sangat tenang.(ini adalah saya versi anime)
